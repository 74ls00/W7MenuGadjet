Thumbnail Database Cleaner enables you to clean up thumbnail cache 
which is used by Windows to speed up the display of thumbnails in 
folders (thumbs.db, ehthumbs.db, thumbcache_*.db files). 
Usually these files are system files and hidden.

If you encounter a problem while running this utility,
please record all the information and send it to:
support@itsamples.com
